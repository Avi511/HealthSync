# 🔔 HealthSync - Notification Service

The Notification Service acts as a message broker in the HealthSync backend, managing transaction receipts, reminders, and patient updates.

---

## ⚙️ Features
* **Alert Dispensation:** Automated routing templates for alert logs.
* **Lightweight Isolation:** Highly optimized Spring Boot service ready to integrate messaging protocols (e.g. SMTP or SMS brokers).

---

## 🔌 Configuration
* **Container Name:** `notification-service`
* **Local Exposed Port:** `8085`
* **Technology Stack:** Spring Boot, Spring Web
